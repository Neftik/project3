# 🧠💥 Распределённый калькулятор арифметических выражений

🚀 **Финальный проект: масштабируемый вычислитель "долгих" выражений**

🔗 Стартовая точка: [http://localhost:8080/](http://localhost:8080/)

---

## 📋 Описание проекта

Пользователь вводит выражение `2 + 2 * 2`, а наш сервис считает его... медленно. Очень медленно.  
Почему? Потому что **в нашей реальности каждое математическое действие требует огромных ресурсов** 🧮⚙️

👉 Поэтому:

- Выражение **разбивается на задачи**
- Задачи раздаются **вычислительным агентам**
- Пользователь получает **ID выражения**
- Потом периодически спрашивает: **"Ну что, посчиталось?"**

---

## 🧩 Архитектура

🔹 **Оркестратор** — сервер, который:
- Принимает выражения
- Разбивает на задачи
- Контролирует очередь задач и собирает результаты

🔹 **Агент** — демон, который:
- Получает задачу от оркестратора
- Выполняет её
- Отправляет результат обратно

## 🧪 Как пользоваться

### 📦 Установка:

```bash
git clone https://github.com/neandrson/go-daev2-final.git
cd go-daev2-final
```

Убедитесь, что у вас установлен [Docker](https://docs.docker.com/get-docker/) (версия 2.24.5+)

### 🚀 Запуск:

```bash
docker compose up -d --scale agent=3
```

✋ Подождите, пока сервер полностью поднимется!

### 🛑 Остановка:

```bash
docker compose down
```

## ➗ Работа с выражениями

### 🧮 Отправка выражения:

```bash
curl --location 'http://localhost:8080/api/v1/calculate' --header 'Content-Type: application/json' --header 'Authorization: Bearer <ваш_токен>' --data '{
  "expression": "2+2*2"
}'
```

### 🔍 Получение списка выражений:

```bash
curl --location 'http://localhost:8080/api/v1/expressions' --header 'Authorization: Bearer <ваш_токен>'
```

### 🔍 Получение по ID:

```bash
curl --location 'http://localhost:8080/api/v1/expression?id=2_p_2' --header 'Authorization: Bearer <ваш_токен>'
```

## 🧪 Дополнительные примеры

### Сложное выражение:

```bash
"expression": "(3 + 4) * (5 - 2) / 3"
```

### Ошибка в выражении:

```bash
"expression": "3 + * 5"
```

👉 Вернёт статус: `invalid`

### Большое выражение:

```bash
"expression": "((10 + 20) * 2 - 5) * (3 + 4 / 2)"
```
